CSS4 Color Names

The CSS4 Color Module includes this set of colornames, 140 of which were originally introduced by the X Windows System. The color 'rebeccapurple' was added in 2014 as a tribute to the late daughter of CSS and web standards leader, Eric Meyer. If you are interested in the history of these whimsical and often illogical names, see this video of a presentation by Alex Sexton.
